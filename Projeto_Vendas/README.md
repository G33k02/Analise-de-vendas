Análise de Vendas

# Objetivo

Este projeto tem como objetivo analisar o desempenho de venda levando alguns fatores em consideração, como regiões,
categorias de produtos, tipos de clientes e metodos de pagamento

# Sobre os dados

O dataset utilizado no projeto tem 1000 registros de vendas

# Tecnologias utilizadas:

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Sql/Sqlite3

# Analises realizadas:

- Faturamento por região
- Metodos de pagamento que mais são utilizados e tambem seu faturamento
- Faturamento por categoria
- Faturamento total e por quantidade das categorias de produtos
- Faturamento por tipos de clientes

# Principais insigths

- A região com maior faturamento indica maior concentração de vendas ou maior ticket médio, sugerindo oportunidades de replicar estratégias em outras regiões.
- Algumas categorias de produtos se destacam tanto em faturamento quanto em volume, indicando maior aceitação pelo mercado.
- Determinados métodos de pagamento são mais utilizados, o que pode orientar decisões operacionais e comerciais.

## O projeto inclui gráficos que facilitam a interpretação dos dados, permitindo identificar rapidamente padrões.

# Conclusão

A análise permitiu identificar padrões relevantes no comportamento de vendas, destacando regiões, produtos e tendencia de vendas ao longo do ano.
